# Local Setup For FitNova

## Current Limitation

Flutter is installed on this machine and the `mobile` folder now contains a generated Android project. The remaining setup is to fetch Dart packages and run the app on an emulator or connected Android device.

## Next Steps

1. Open a terminal in `C:\Users\alokm\OneDrive\Desktop\FIT APP 001\mobile`
2. Run:

```powershell
flutter pub get
```

3. Open `C:\Users\alokm\OneDrive\Desktop\FIT APP 001\mobile` in Android Studio
4. Install Flutter and Dart plugins in Android Studio if prompted
5. Start an emulator or connect a device
6. Run the app
